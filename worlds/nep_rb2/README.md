# Hyperdimension Neptunia Re;Birth 2 Sisters Generation

## What does Randomization do to this game?


Randomization will begin after players enter the guild event where Nepgear, IF and Compa go to the guild to accept quests.
Things randomized will be enemies, gathers and treasures, and hidden treasures. Colosseum battles, guild quests.

## What is the goal of HDN RB2?

There will be several goals for RB2.
True End True Arfoire
Conquest Ending
Bounty Hunting

## What items and locations can get shuffled?

- All Items, Weapons, Armors, Ornaments, Clothing, plans
- All Treasure, Gather points, Enemies, Colosseum rewards, guild rewards.
- All Dungeon unlocks
