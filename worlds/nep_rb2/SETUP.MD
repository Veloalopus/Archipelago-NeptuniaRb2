## Before Playing

The randomizer assumes you've completed the base game at the very least.
It is recommended to have played the base game and finished a True Ending playthrough.

## To Install

1. Download and install the latest Archipelago release on the [Archipelago Releases Page](https://github.com/ArchipelagoMW/Archipelago/releases)
2. Download the 'nep_rb2.apworld' from the latest release on [Archipelago HDN RB2 Release Page](https://github.com/Veloalopus/Archipelago-NeptuniaRb2/releases)
3. Open 'nep_rb2.apworld' with the Archipelago Launcher, is should automatically install.
4. If it does not automatically install, navigate to your Archipelago worlds folder (by default, `C:\ProgramData\Archipelago\lib\worlds) and place the ap world there.
5. Download the zip Archipelago Neptunia Client from [BlackSoulKnight Releases Page](https://github.com/BlackSoulKnight/Neptunia_ReBirth_3_Archipelago_Client/releases)
5. Download Reloaded II from [Reloaded II Releases Page](https://github.com/Reloaded-Project/Reloaded-II/releases)


## To Generate a Game

1. Download 'nep_rb2.yaml' from the latest release on the [Archipelago HDN RB2 Releases page](https://github.com/Veloalopus/Archipelago-NeptuniaRb2/releases)
2. Edit the file to your desired settings.
3. Navigate to your Archipelago Installation (`C:\ProgramData\Archipelago)
4. Navigate to the 'Players' folder under your base folder.
5. Move your 'nep_rb2.yaml' into this folder.
6. Open the Archipelago Launcher and click the **GENERATE** button.
7. Your generated game will be in the 'Output' folder in the Archipelago base folder.

## To Host a Game

1. Navigate to the [Host Game](https://archipelago.gg/uploads) page.
2. Upload the '.zip' file outputted from the above "To Generate a Game"

## To Start Playing

1. Open Reloaded II, click on the Add an Application button, find your NeptuniaRebirth2.exe, and select that.
2. Go to the tab for Neptunia Rebirth 2, and drag/drop the ZIP folder from step 5 of TO INSTALL into the Reloaded II window.
3. Click on the mod and make sure it has a PLUS + sign, then configure the mod for your port, room and player name, then click save.
4. Launch the game through Reloaded II, and you should automatically connect through with the console!
5. Select New Game at title screen. It is recommended to have a save right after you first gain control and are able to move around the overworld, and load that instead. (Saves 2-3 minutes of waittime)
6. IT is recommended to play with a Universal Tracker and maybe another text client, as the reloaded II console cannot be used to function as a console for AP world commands. (Hints, etc)
